INDOOR RUN

暑い日も安心。
冷房完備の室内ドッグラン

天候に左右されずに楽しめる、50㎡超の室内ドッグラン。 暑くなってきたら、冷房完備の室内へ移動してゆっくり遊べます。

広々とした室内には水道も完備。 毎月変わる壁面デコレーションや顔ハメパネルで、 記念撮影もお楽しみください。
